# Онлайн-чат (сайт ↔ Telegram)

Клієнт пише у віджет чату на сайті → повідомлення приходить у Telegram.
Ваша відповідь у Telegram миттєво з'являється у клієнта на сайті.

## Дві «кімнати»

- **Кімната замовлень** — особисті повідомлення бота (як і було): туди приходять замовлення з кошика.
- **Кімната чату** — окрема Telegram-група з Темами (Topics): кожен клієнт = окрема тема. Відповідаєте, просто написавши в тему — без Reply і хештегів. Бот ставить 👍, коли відповідь доставлена клієнту.

## Налаштування кімнати чату (один раз)

1. **SQL**: відкрийте https://supabase.com/dashboard/project/vhvedefyixgluayqahhh/sql/new →
   вставте вміст файлу [supabase/chat_rooms.sql](supabase/chat_rooms.sql) → **Run**.
2. Створіть у Telegram нову **групу**, наприклад «AUTOSHOP чат», і додайте туди бота.
3. Налаштування групи → увімкніть **Теми** (Topics).
4. Зробіть бота **адміністратором** групи з правом **«Керування темами»** (Manage Topics).
5. Надішліть у групу команду `/setup` — бот відповість «✅ Готово!».

Після цього кожне нове звернення з сайту створює тему «💬 Клієнт a1b2c3d4», і вся переписка з цим клієнтом живе в ній.

Якщо кімната чату не налаштована, працює запасний режим: повідомлення приходять в особисті
повідомлення бота з тегом `#chat_...`, відповідати треба через Reply.

## Технічно

1. Віджет ([src/components/ChatWidget.tsx](src/components/ChatWidget.tsx)) відправляє повідомлення на `/api/chat-send`.
2. [api/chat-send.mjs](api/chat-send.mjs) зберігає його в Supabase (`chat_messages`) і пересилає в тему клієнта в групі (`chat_sessions` зберігає відповідність сесія↔тема, `chat_config` — id групи).
3. Ваше повідомлення в темі приходить на webhook [api/telegram-webhook.mjs](api/telegram-webhook.mjs), який зберігає відповідь у Supabase.
4. Віджет підписаний на Supabase Realtime — відповідь з'являється у клієнта миттєво, історія зберігається.

Webhook зареєстровано на `https://autoshop-market.vercel.app/api/telegram-webhook`.
Перереєстрація (якщо зміниться домен):

```
https://api.telegram.org/bot<ТОКЕН_БОТА>/setWebhook?url=https://<ВАШ_ДОМЕН>/api/telegram-webhook&secret_token=autoshop_chat_hook_x9K2mQ7pL4vR8sT1
```

## Локальна розробка

`npm run dev` показує віджет, але `/api/*` функції працюють лише на Vercel
(або через `vercel dev`). Повний цикл перевіряйте на задеплоєному сайті.
