# Music Player

Простой нативный Android-плеер на Kotlin. **Без рекламы, без трекинга, без интернета** —
приложение вообще не запрашивает доступ к сети. Оно играет музыку, которая уже есть на
вашем устройстве.

## Возможности

- **Просмотр по папкам** — музыка сгруппирована по папкам устройства; заходите в папку и видите её треки
- **Воспроизведение из конкретной папки** — включённый трек задаёт очередь именно из этой папки
- **Случайный порядок (shuffle)** — кнопка перемешивает треки в пределах текущей папки; текущий трек не прерывается
- Список треков с названием, исполнителем и длительностью
- Воспроизведение / пауза / следующий / предыдущий трек
- Перемотка (seek bar) с отображением времени
- **Фоновое воспроизведение** — музыка играет, когда приложение свёрнуто
- Управление из шторки уведомлений и с экрана блокировки (MediaSession)
- Корректная работа с аудиофокусом (пауза при звонке, приглушение при уведомлениях)
- Тёмная тема

## Требования

- Android Studio (Hedgehog или новее)
- Android SDK 34
- Минимальная версия Android на телефоне: 7.0 (API 24)

## Как собрать APK

1. Откройте папку `music-player/` в **Android Studio** (`File → Open`).
2. Дождитесь, пока Gradle синхронизирует зависимости (нужен интернет только на этом шаге).
3. Подключите телефон по USB с включённой отладкой **или** используйте эмулятор.
4. Нажмите **Run ▶** — приложение соберётся и установится.

Чтобы получить APK-файл для установки вручную:
`Build → Build Bundle(s) / APK(s) → Build APK(s)`.
Готовый файл появится в `app/build/outputs/apk/debug/app-debug.apk`.

> При первом открытии Android Studio сама создаст Gradle wrapper, если его нет.
> Если собираете из командной строки, сначала выполните `gradle wrapper`, затем
> `./gradlew assembleDebug`.

## Права доступа

Приложение запрашивает только:

- **Доступ к аудиофайлам** (`READ_MEDIA_AUDIO` / `READ_EXTERNAL_STORAGE`) — чтобы найти музыку.
- **Уведомления** (`POST_NOTIFICATIONS`) — для панели управления плеером.
- **Фоновый сервис** (`FOREGROUND_SERVICE`) — чтобы музыка не прерывалась при сворачивании.

Доступа к интернету нет — реклама показываться физически не может.

## Структура

```
app/src/main/
├── java/com/example/musicplayer/
│   ├── MainActivity.kt          # экран: папки → треки + панель управления
│   ├── MusicService.kt          # фоновый сервис воспроизведения + shuffle (ядро)
│   ├── MediaStoreRepository.kt  # чтение музыки и группировка по папкам
│   ├── FolderAdapter.kt         # адаптер списка папок
│   ├── SongAdapter.kt           # адаптер списка треков
│   ├── MusicFolder.kt           # модель папки
│   └── Song.kt                  # модель трека
└── res/                         # макеты, цвета, тема, иконки
```
