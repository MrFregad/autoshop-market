## Кроки налаштування (по порядку!)

### 1. Додати service-ключ Supabase на Vercel

1. https://supabase.com/dashboard/project/vhvedefyixgluayqahhh/settings/api → скопіюйте ключ **service_role** (секретний!).
2. https://vercel.com → проєкт autoshop-market → **Settings → Environment Variables** →
   додайте: ім'я `SUPABASE_SERVICE_KEY`, значення — скопійований ключ.

### 2. Перевипустити токен бота

1. У Telegram відкрийте **@BotFather** → `/revoke` → оберіть бота → скопіюйте **новий** токен.
   (Старий токен «засвічений» у старих версіях сайту — його треба анулювати.)
2. На Vercel додайте env-змінну `TELEGRAM_BOT_TOKEN` з новим токеном.

### 3. Задеплоїти

Запушити зміни в git (Vercel підхопить env-змінні при деплої).

### 4. Закрити таблиці в базі

https://supabase.com/dashboard/project/vhvedefyixgluayqahhh/sql/new →
вставити вміст [supabase/secure_tables.sql](supabase/secure_tables.sql) → **Run**.
(Тільки ПІСЛЯ кроків 1-3, інакше адмінка тимчасово не зможе зберігати товари.)

### 5. Перереєструвати webhook чату (новий токен!)

Відкрити в браузері (підставивши новий токен):

```
https://api.telegram.org/bot<НОВИЙ_ТОКЕН>/setWebhook?url=https://autoshop-market.vercel.app/api/telegram-webhook&secret_token=autoshop_chat_hook_x9K2mQ7pL4vR8sT1
```

Має відповісти `{"ok":true,...,"description":"Webhook was set"}`.

## Перевірка після налаштування

- Оформіть тестове замовлення на сайті → воно має прийти в Telegram.
- Напишіть у чат на сайті → повідомлення має прийти в кімнату чату.
- Введіть новий пароль у пошук на сайті → має відкритися адмінка; спробуйте зберегти товар.
